Global Pilot — EPS Logo Concept Set

These EPS files are vector artwork with no background rectangle or fill. In EPS terms, that means the canvas area around the mark is transparent when placed into Illustrator, InDesign, Keynote, PowerPoint, Figma, or other design tools that support EPS import.

Included concepts:
1. CompassGlobe — app icon using a globe grid, active flight route, and navigation arrow.
2. RouteHorizon — rounded app-icon form with horizon/globe arcs and an outbound route.
3. WeatherNav — weather + compass/nav visual for flight planning, weather, and navigation.
4. MonogramWing — compact GP monogram with wing/route motion.
5. WordmarkOnly — outlined vector wordmark reading Global Pilot.
6. StackedBadge — icon + wordmark in a compact vertical badge.

Production notes:
- Text has been converted to vector outlines; no font file is required.
- EPS does not use alpha-channel transparency like PNG. Transparency is achieved by not drawing a background object.
- Recommended refinement path: choose the strongest concept, then tighten spacing/kerning and color system in Illustrator/Figma for the final app-store production package.
